# 다른 여정으로 이동

한 여정에서 다른 여정으로 개인 사용자를 푸시하는 방법을 알아봅니다.

Jump 작업 활동을 통해 다음을 수행할 수 있습니다.

* 여러 개로 분할하여 매우 복잡한 여정의 디자인을 간소화
* 공통 및 재사용 가능한 여정 패턴을 기반으로 여정 구축
